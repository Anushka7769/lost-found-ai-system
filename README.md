# AI-Based Lost and Found Matching System

## Project Overview

This is a web-based Lost and Found Management System developed for the University of Peradeniya. The system helps students recover lost items efficiently using centralized reporting, AI-based matching, and Telegram integration.

The system allows students to report lost items and allows anyone (students, staff, visitors, or workers) to report found items. The system automatically matches lost and found items using Artificial Intelligence based on text and image similarity.

The system also integrates with Telegram to automatically share item advertisements in the official Lost and Found group. Users can also manually share advertisements using their own WhatsApp accounts.

The system is designed as a responsive web application that works on mobile phones, tablets, and desktop computers.

---

## Key Features

- Student registration and secure login using university registration number
- Lost item reporting (students only)
- Found item reporting (anyone without registration)
- Image upload support
- AI-based matching using text similarity (Phase 1)
- AI-based matching using image similarity (Phase 2)
- Automatic advertisement posting to Telegram group using Telegram Bot
- Manual advertisement sharing via WhatsApp
- Faculty-based filtering system
- Notification system for match suggestions
- Admin panel for managing posts
- Automatic deletion of old records after one month
- Mobile-friendly responsive design

## System Users

### Student Users
- Register using university registration number and mobile number
- Report lost items
- Report found items
- Upload item images
- View lost and found items
- Receive match notifications
- Share advertisements via WhatsApp
- Enable Telegram sharing

### Guest Users
- Report found items without registration
- Upload images and enter item details
- Provide contact information

### Administrator
- View all system data
- Remove invalid or fake posts
- Manage system and Telegram integration
